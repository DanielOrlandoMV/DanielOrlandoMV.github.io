Veterinaria

Este proyecto fue realizado a nivel de código sin contar con el uso de formularios
o alguna otra herramienta visual, este sistema esta enfocado en una veterinaria
donde se pueden realizar registros de mascotas, dueños y ventas las cuales tiene una estructura
de PILA y COLA, listados simplemente enlazada, doblemente enlazada y lista circular además
de contar con un apartado donde se tiene el ordenamiento de árbol ABB y AVL.
