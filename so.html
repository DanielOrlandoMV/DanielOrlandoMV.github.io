Este proyecto desarrollado en Visual Studio C++ se creó como un controlado de procesos
y controlador de apartados de importancia sobre el sistema operativo y hardware
contamos con una pantalla donde se encuentran los distintos procesos basicos
que gestiona como identificar datos del SO, Hardware, Redes, también el controlador de procesos
que permite a los usuarios crear, gestionar y monitorear procesos
el controlador de memoria se encarga de la asignación y liberación de memoria para dichos procesos.
