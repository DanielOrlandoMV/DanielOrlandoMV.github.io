Analizador Lexico C++

El desarrollo del siguiente analizador léxico fue realizado con Visual Studio C++
el analizador es una herramienta para el proceso de análisis de lenguaje de programación y procesamiento de texto
su principal función es la de transformar una serie de caracteres como se puede apreciar en la pantalla y pasarlo
a una secuencia de tokens que puede ser manejado de manera fácil por otro analizador que sería el analizador sintáctico.
